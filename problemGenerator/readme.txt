Tableau Prover Problem Generator

The programs in this package generate four classes of problems:
gamma, php, h and statman.

Instructions for instalation:
1. Run install.sh in the problemGenerator folder

The generated problems will be placed in the directory indicated in the bin/createAll.sh file.

In order to change the problems created, change the attributes in the .sh
files in the bin folder.

In order to clean unnecessary files for compactation, run clean.sh.
